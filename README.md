# Vaultique 🔒🎒

Vaultique is a software engineering project that introduces a digital platform for preserving memories by connecting physical keepsakes with digital content. The system organizes memories into structured life chapters using QR code integration.

---

## 👑 My Role: Team Leader & Core Contributor

This project was a collaborative effort developed by a group of four student engineers. As the **Team Leader and Project Organizer**, I managed the structural team pipelines, synchronized our cross-modular milestones, and held complete ownership over the comprehensive technical delivery format. 

Alongside my leadership responsibilities, I took direct ownership of the following specific research, engineering, and design modules:

### 🛠️ Core Technical & Design Implementations
* **Project Initiation & Architecture:** Formulated the primary baseline of the project, including the core Scope of Work and the comprehensive baseline Project Idea & Justification.
* **Problem Scoping:** Engineered and authored the core Problem Statement, defining the parameters of "Memory Fragmentation" across digital and physical spaces.
* **User Analysis:** Independently executed and drafted the entire User Analysis layer, building out our stakeholder personas and structural target-user use cases.
* **UI Wireframe Prototyping:** Designed and constructed the functional Web UI Wireframe prototypes, mapping out interface navigation frameworks for the platform's core layouts.
* **Quality Verification:** Structured the foundational testing parameters by implementing core test scenarios and validation test cases to initiate our quality assurance pipeline.
